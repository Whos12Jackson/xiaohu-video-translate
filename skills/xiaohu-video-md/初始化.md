# video skill 初始化指南

本文档仅在首次使用或依赖缺失时由 AI 读取，用于自动安装和配置所需依赖。

## 自动检测与安装

**首次使用时，AI 应自动检测并安装缺失的依赖，无需用户干预。**

### 依赖检测

检测命令：
```bash
# 检查工具
which yt-dlp ffmpeg whisper-cli

# 检查模型
ls ~/.local/share/whisper/ggml-large-v3-turbo.bin 2>/dev/null
ls ~/.local/share/whisper/ggml-medium.bin 2>/dev/null
```

### 自动安装流程

**1. 检测到缺失工具时**：
```bash
# 安装 Homebrew（如果未安装）
if ! command -v brew &> /dev/null; then
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
fi

# 安装核心工具
brew install yt-dlp ffmpeg whisper-cpp
```

**注意**：使用 `--prompt "以下是简体中文的句子。"` 参数引导 Whisper 直接输出简体中文。

**2. 检测到缺失模型时**：
```bash
# 创建模型目录
mkdir -p ~/.local/share/whisper
cd ~/.local/share/whisper

# 下载 large-v3-turbo 模型（如缺失）
[ ! -f ggml-large-v3-turbo.bin ] && \
  curl -L -o ggml-large-v3-turbo.bin \
  "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-large-v3-turbo.bin?download=true"

# 下载 medium 模型（如缺失）
[ ! -f ggml-medium.bin ] && \
  curl -L -o ggml-medium.bin \
  "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-medium.bin?download=true"
```

### 首次使用流程（完全自动化）

1. 用户请求视频处理
2. AI 检测依赖 → 发现缺失
3. AI **读取本文档并静默自动安装**（输出进度信息）
4. 安装完成 → 立即执行用户请求
5. 全程无需用户操作

### 依赖清单

- **工具**（通过 Homebrew 安装）：
  - `yt-dlp` - 视频下载
  - `ffmpeg` - 音视频处理
  - `whisper-cpp` - 音频转写（Apple Silicon 优化）

- **模型**（两个核心模型必需，共 2.9GB）：
  - `ggml-large-v3-turbo.bin`（精确模式，1.5GB）
  - `ggml-medium.bin`（快速模式，1.4GB）

### 注意事项

- 安装过程中输出简要进度信息，让用户了解状态
- macOS 以外系统可能需要调整安装命令
- 受限资源时，先索要必要凭据（cookies/代理）再尝试
- YouTube 如遇 403/SABR/PO Token：优先用 `yt-dlp --cookies-from-browser chrome ...`（需要本机已安装对应浏览器且能读取其 cookies）

### 安装完成验证

```bash
# 验证工具
which yt-dlp ffmpeg whisper-cli

# 验证模型
ls -lh ~/.local/share/whisper/
```

全部检测通过后，返回主流程继续执行用户请求。
